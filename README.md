# Description

Intro to Rails Project (Assignment 3) for Agile Full Stack Web Development

This project is a website that lists out dog breeds with images and reviews, helping potential future or current dog owners know which dog is best to adopt based on personality and ease of care.

DOG CEP API
* https://dog.ceo/api/breeds/list/all
provides a list of real dog breeds
* /api/breed/{name}/images/random
provides a real photo of each dog
* Faker GEM generates fake reviews

* Tables & ERD

Breeds stores the dog breed names and a photo URL of what the dog breed looks like.
Reviews stores user reviews of each breed, fake information is genereated by Faker. Each review includes a rating, comment and trait scores.
