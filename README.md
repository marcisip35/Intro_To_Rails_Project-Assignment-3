# Description

Intro to Rails Project (Assignment 3) for Agile Full Stack Web Development

This project is a website that lists out dog breeds with images and reviews, helping potential future or current dog owners know which dog is best to adopt based on personality and ease of care.

## Data Sources
DOG CEO API
* https://dog.ceo/api/breeds/list/all
provides a list of real dog breeds
* /api/breed/{name}/images/random
provides a real photo of each dog
* Faker GEM generates fake reviews

* The seven breed groups are hardcoded in the seed script.

## Tables

Breeds stores the dog breed names and a photo URL of what the dog breed looks like.
Groups stores the dog breed groups.
Reviews stores user reviews of each breed, generated by Faker. Each review includes a rating, comment, and three trait scores.

## Relationships

A group has many breeds, and a breed has many reviews. 
The seed script pulls the breed list from the API first, assigns each breed a random group, then generates two to four Faker reviews for every breed. 
This produces over 400 rows across the three tables.
